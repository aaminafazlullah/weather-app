# User Stories

## Weather App

- As a user, I want to enter a city name so that I can see the current weather.
- As a user, I want to see the temperature so that I know how hot or cold it is outside.
- As a user, I want to see the humidity so that I know how humid it is.
- As a user, I want to see the weather condition so that I know what to expect outside.
- As a user, I want to see an error message if I enter an invalid city so that I know to try again.
- As a user, I want to access the app from a browser so that I don't need to install anything.
